# 🎮 Кто хочет стать миллионером?
## Python + tkinter

### Структура проекта
```
millionaire/
├── main.py                  ← ЗАПУСКАТЬ ОТСЮДА
├── settings/
│   ├── config.py            ← настройки сложностей
│   └── storage.py           ← сохранение баланса (save.json)
├── background/
│   └── README_bg.txt        ← инструкция по фону
├── questions/
│   └── questions.py         ← ВСЕ ВОПРОСЫ ЗДЕСЬ
├── answers/
│   └── answers.py           ← кнопки A/B/C/D
└── game/
    └── engine.py            ← игровой движок
```

### Запуск
```bash
python main.py
```
Требуется Python 3.10+ с модулем tkinter (входит в стандартную установку).

### Как добавить свои вопросы
Открой `questions/questions.py` и замени строки вида:
```python
{"q": "Default Text — Лёгкий вопрос 1?", "a": ["Ответ A","Ответ B","Ответ C","Ответ D"], "correct": 0},
```
на настоящие вопросы. `correct` — индекс правильного ответа (0=A, 1=B, 2=C, 3=D).

### Как добавить фоновое изображение
В `main.py` найди метод `show_menu()` и раскомментируй блок:
```python
# from PIL import Image, ImageTk
# img = ImageTk.PhotoImage(Image.open("background/bg.jpg").resize((900,650)))
# ...
```
Установи Pillow: `pip install Pillow` и положи картинку в `background/bg.jpg`.

### Механики
| Режим       | Вопросов | Жизней | Награда за вопрос | Разблокировка |
|-------------|----------|--------|-------------------|---------------|
| Лёгкий      | 10       | ❤❤❤    | +$500 000         | бесплатно     |
| Средний     | 20       | ❤❤     | +$1 000 000       | $5 000 000    |
| Тяжёлый     | 30       | ❤      | +$2 500 000       | $15 000 000   |
| Бесконечный | ∞        | ❤❤     | смешанные         | бесплатно     |

- **Выйти и сохранить** — покидаешь игру, деньги остаются
- **Проиграл (0 жизней)** — GAME OVER, теряешь ВСЕ деньги
- **Победа** — ответил на все вопросы, показывается итоговый баланс
